# 心遇伴行 · AI 一对一剧情互动 Demo

基于《直播 AI 一对一剧情互动功能【心遇伴行】V1.0 产品需求文档 PRD》制作的轻量化可运行 demo，用于产品体验与内部评审。

## 在线预览

🌐 GitHub Pages: https://2477304427gs-rgb.github.io/xinyu-banxing-demo/

## 本地运行

无需构建工具，直接用浏览器打开 `index.html` 即可。

```bash
# 或者启动一个本地 HTTP 服务器
npx serve .
```

推荐使用 Chrome DevTools 的手机模式预览，以获得最佳竖屏体验。

## 已实现功能

- **双页面互斥**：真人直播页与 AI 剧情互动页一键切换，永不共同出镜。
- **三位主播**：幼甜、发升米、锦汐 Sisi，每位拥有独立的性格 Skill、主题色与推荐剧情。
- **AI 二次元分身**：风格化 SVG 占位女主，带呼吸、眨眼动画。
- **三大剧情类目**：现代治愈羁绊、古风宿命偏爱、次元幻境羁绊，每条剧情含多分支结局。
- **羁绊好感体系**：单主播独立累计，四个等级，正向/逆向选择影响好感进度。
- **全自动互动**：进入剧情页后 AI 自主推进，底部二选一决策。
- **语音模拟**：使用浏览器 Web Speech API 按情绪朗读台词。
- **商业化入口**：演示会员、限定剧情包、AI 人设定制等 UI 入口。
- **主播端管控**：功能总开关、清空队列、人设锁定模拟。

## 体验路径

1. 打开页面，默认进入「幼甜」的真人直播间。
2. 点击右侧悬浮「心遇伴行」入口，模拟排队后进入 AI 剧情页。
3. 阅读女主台词，在底部选择分支选项，观察羁绊变化。
4. 点击左上角退出返回直播间，或等待 30 秒无操作自动结束。
5. 点击右上角人像图标可切换主播，每位主播剧情与反馈不同。

## 说明

- 三位主播的二次元分身已使用 AI 生成的专属形象（位于 `assets/ai/`），底部直播 UI 通过渐变遮罩处理。
- 语音为浏览器 TTS 占位演示，非真实克隆音色。
- 商业化入口仅做 UI 示意，无真实支付能力。
- 详见 [`PROMPTS.md`](./PROMPTS.md) 了解 AI 图片生成提示词与替换方式。
